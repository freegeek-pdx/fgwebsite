Plugin used to overlay images on the current page into neat Javascript-powered overlay popups.

== Description ==

This plugin includes the new Lightbox JS v2.04 javascript written by Lokesh Dhakar and got transformed into a Wordpress Plugin by me.
Use the title attribute if you want to show a caption.
[Plugin Homepage](http://www.4mj.it/lightbox-js-v20-wordpress/ "Plugin Homepage")

== Installation ==

1. Upload `lightbox.php` and "lightbox folder" to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a rel="lightbox" attribute to any link tag to activate the lightbox or rel="lightbox[roadtrip]" for image set

If you have already installed the plugin:

1. De-activate the plugin.
2. Delete the old version (lightbox.php and lightbox folder), upload the new version.
3. Re-activate the plugin.
